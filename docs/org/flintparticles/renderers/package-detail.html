Renderers are used to draw the particle system on the stage.


Interfaces
 InterfaceDescription
 Renderer The Renderer interface must be implemented by all renderers.
Classes
 ClassDescription
 BitmapRenderer The BitmapRenderer draws particles onto a single Bitmap display object.
 DisplayObjectRenderer The DisplayObjectRenderer adds particles to its display list and lets the flash player render them in its usual way.
 FullStageBitmapRenderer [Deprecated] The FullStageBitmapRenderer is a custom BitmapRenderer whose canvas exactly covers the stage.
 FullStagePixelRenderer [Deprecated] The FullStagePixelRenderer is a custom PixelRenderer whose canvas exactly covers the stage.
 PixelRenderer The PixelRenderer draws particles as single pixels on a Bitmap display object.
