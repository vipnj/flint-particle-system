Packageorg.flintparticles.initializers
Classpublic class Position
ImplementsInitializer

The Position Initializer sets the initial location of the particle.

The class uses zones to place the particle. A zone defines an area of the stage and the particle is placed at a random point within that area. For precise placement, the Point zone defines a single point at which all particles will be placed. Various zones (and the Zones interface for use when implementing custom zones) are defined in the org.flintparticles.zones package.
