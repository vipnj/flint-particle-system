Classpublic class FirstPersonCamera
InheritanceFirstPersonCamera KeyboardControllerBase

Sets keyboard input to make a renderer's camera track, pan and tilt in response to keyboard input.

After assigning a camera to an instance of this class, the camera responds to the following keyboard input

  • Up arrow key - tilt down.
  • Down arrow key - tilt up.
  • Left arrow key - pan left.
  • Right arrow key - pan right.
  • W key - track in.
  • S key - track out.
  • A key - track left.
  • D key - track right.
  • Page up key - raise the camera.
  • Page down key - lower the camera.

public function FirstPersonCamera(stage:DisplayObject = null, camera:Camera = null, rotationRate:Number = 1, trackRate:Number = 200, useInternalTick:Boolean = true)

The constructor creates an FirstPersonCamera controller.

Parameters
stage:DisplayObject (default = null) — The display object on which to listen for keyboard input. This should usually be a reference to the stage.
 
camera:Camera (default = null) — The camera to control from the keyboard.
 
rotationRate:Number (default = 1) — The rate at which to rotate the camera when the appropriate keys are pressed, in radians per second.
 
trackRate:Number (default = 200) — The rate at which to track the camera when the appropriate keys are pressed, in units per second.
 
useInternalTick:Boolean (default = true) — Indicates whether the camera controller should use its own tick event to update its state. The internal tick process is tied to the framerate and updates the camera every frame.
