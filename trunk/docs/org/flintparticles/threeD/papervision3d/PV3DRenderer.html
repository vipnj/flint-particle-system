Classpublic class PV3DRenderer
InheritancePV3DRenderer RendererBase

Renders the particles in a Papervision3D scene.

To use this renderer, the particles' image properties should be Papervision3D objects, renderable in a Papervision3D scene. This renderer doesn't update the scene, but copies each particle's properties to its image object so next time the Papervision3D scene is rendered the image objects are drawn according to the state of the particle system.

addParticle()method
protected override function addParticle(particle:Particle):void

This method is called when a particle is added to an emitter - usually because the emitter has just created the particle. The method adds the particle's image to the container's display list. It is called internally by Flint and need not be called by the user.

Parameters
particle:Particle — The particle being added to the emitter.
