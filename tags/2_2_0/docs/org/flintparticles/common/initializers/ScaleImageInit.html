The ScaleImageInit Initializer adjusts the size of the particles image.

If you also want to adjust the mass and collision radius of the particle, use the ScaleAllInit initializer.

Constructor detail
ScaleImageInit()constructor
public function ScaleImageInit(minScale:Number = 1, maxScale:Number)

The constructor creates a ScaleImageInit initializer for use by an emitter. To add a ScaleImageInit to all particles created by an emitter, use the emitter's addInitializer method.

The scale factor of particles initialized by this class will be a random value between the minimum and maximum values set. If no maximum value is set, the minimum value is used with no variation.

Parameters
minScale:Number (default = 1) — the minimum scale factor for particles initialized by the instance.
 
maxScale:Number — the maximum scale factor for particles initialized by the instance.
